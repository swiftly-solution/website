Adds an extra button to the menu that executes a custom action when pressed.

```csharp
IMenuBuilderAPI AddExtraButton(KeyBind keyBind, string label, Action<IPlayer, IMenuAPI> action)
```

<ApiLabel>Parameters</ApiLabel>

- <ApiParam name="keyBind" type="KeyBind" typeHref="/docs/api/menus/keybind" /> — The key binding for this button.
- <ApiParam name="label" type="string" typeHref="https://learn.microsoft.com/dotnet/api/system.string" /> — The label to display for this button in the menu footer.
- <ApiParam name="action" type="Action&lt;IPlayer, IMenuAPI&gt;" typeHref="https://learn.microsoft.com/dotnet/api/system.action-2" /> — The action to execute when the button is pressed.

<ApiLabel>Returns</ApiLabel>

- <ApiParam type="IMenuBuilderAPI" typeHref="/docs/api/menus/imenubuilderapi" /> — This builder for method chaining.

<ApiLabel>Remarks</ApiLabel>

Extra buttons are displayed in the menu footer and allow custom actions beyond standard menu navigation.
The keybind should not conflict with existing menu navigation keys.

### SetExitButton(KeyBind)

<ViewSource href="https://github.com/swiftly-solution/swiftlys2/blob/master/managed/src/SwiftlyS2.Shared/Modules/Menus/IMenuBuilderAPI.cs#L114" />

Overrides the default key binding for closing the menu.

```csharp
IMenuBuilderAPI SetExitButton(KeyBind keyBind)
```

<ApiLabel>Parameters</ApiLabel>

- <ApiParam name="keyBind" type="KeyBind" typeHref="/docs/api/menus/keybind" /> — The key binding to use.

<ApiLabel>Returns</ApiLabel>

- <ApiParam type="IMenuBuilderAPI" typeHref="/docs/api/menus/imenubuilderapi" /> — This builder for method chaining.

<ApiLabel>Remarks</ApiLabel>

This overrides the default exit button behavior.
Supports multiple key bindings using the bitwise OR operator.
Example: `KeyBind.Esc | KeyBind.A` allows either Esc or A to close the menu.
