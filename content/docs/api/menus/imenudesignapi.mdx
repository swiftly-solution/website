
<ApiLabel>Parameters</ApiLabel>

- <ApiParam name="count" type="int" typeHref="https://learn.microsoft.com/dotnet/api/system.int32" /> — The maximum visible item count. Valid range is 1-5. Default is 5.

<ApiLabel>Remarks</ApiLabel>

Menus with more options than this limit will be paginated.
