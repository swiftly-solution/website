# API Reference

Welcome to the SwiftlyS2 API Reference documentation. This section provides comprehensive documentation for all classes, interfaces, methods, and properties available in the SwiftlyS2 framework.

## Overview

SwiftlyS2 is a powerful framework for Counter-Strike 2 server-side scripting and plugin development. The API is organized into several namespaces that provide different functionalities.
