---
title: OptionsBase
---

- [ButtonMenuOption](/docs/api/core/menus/optionsbase/buttonmenuoption) - Represents a clickable button menu option.
- [ChoiceMenuOption](/docs/api/core/menus/optionsbase/choicemenuoption) - Represents a menu option that cycles through a list of choices.
- [InputMenuOption](/docs/api/core/menus/optionsbase/inputmenuoption) - Represents a menu option that allows text input from players.
- [MenuOptionBase](/docs/api/core/menus/optionsbase/menuoptionbase) - Provides a base implementation for menu options with event-driven behavior.
- [ProgressBarMenuOption](/docs/api/core/menus/optionsbase/progressbarmenuoption) - Represents a progress bar menu option that displays progress visually.
- [SliderMenuOption](/docs/api/core/menus/optionsbase/slidermenuoption) - Represents a slider menu option that allows selecting a numeric value within a range.
- [SubmenuMenuOption](/docs/api/core/menus/optionsbase/submenumenuoption) - Represents a menu option that opens a submenu when clicked.
- [TextMenuOption](/docs/api/core/menus/optionsbase/textmenuoption) - Represents a simple text-only menu option without interactive behavior.
- [ToggleMenuOption](/docs/api/core/menus/optionsbase/togglemenuoption) - Represents a toggleable menu option that displays an on/off state.
